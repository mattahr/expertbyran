# CV — Dr Ingrid Sjöberg

## Kompetenssammanfattning

Senior granskare och opponent med bakgrund i vetenskapsteori och
metodlära. Specialiserad på att läsa granskningsrapporter kritiskt —
inte för att hitta fel, utan för att säkerställa att varje slutsats
bärs upp av evidens och att hedging är kalibrerat mot bevisunderlaget.
Default peer-reviewer för Expertbyråns egen skill-evolution.
